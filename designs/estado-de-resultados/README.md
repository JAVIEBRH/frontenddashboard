# Diseño Estado de Resultados - Figma

Esta carpeta contiene los diseños de Figma para el componente "Estado de Resultados".

## Cómo subir el diseño desde Figma

1. **Exportar desde Figma:**
   - Abre el diseño en Figma
   - Ve a `File > Export` o selecciona los elementos y usa `Export` en el panel derecho
   - Exporta como:
     - **PNG/SVG** para imágenes
     - **PDF** para el diseño completo
     - O usa `File > Save as > Copy link` para obtener el link de Figma

2. **Guardar archivos:**
   - Coloca los archivos exportados en esta carpeta
   - Nombra los archivos de forma descriptiva (ej: `estado-resultados-donut-chart.png`)

3. **Subir a Git:**
   ```bash
   git add designs/estado-de-resultados/
   git commit -m "Add Figma design for Estado de Resultados"
   git push
   ```
